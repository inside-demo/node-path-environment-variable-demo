# node-path-environment-variable-demo

> Test NODE_PATH environment variable

## Run demo

- Clone this repo
- Install packages
  ```shell
  npm install
  ```
- Run script
  ```shell
  npm start
  ```
- See outputs
  ```
  NODE_ENV: development
  NODE_PATH: ./utils;./node_modules
  0.3
  0.02
  0.4472135954999579
  ```
